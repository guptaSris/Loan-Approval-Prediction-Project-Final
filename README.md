# Loan-Approval-Prediction-Project-Final
Built an ML classification model to predict Loan_Status using applicant features. 
Performed preprocessing including missing value imputation, IQR-based outlier capping, log transformation, feature encoding, and scaling.
Trained Logistic Regression and Random Forest models, evaluated using Precision, Recall, F1-score and Cross-Validation, optimizing for reduced false approvals.
